## What is Fraig?
Fraig(Functionally Reduced And-Inverter Graph), is a tool that can reduce redundant logic gates in a circuit

## Getting started
Before getting started, clone the whole file into your computer, and type the following words

```
$ make clean
$ make
$ ./fraig
```
